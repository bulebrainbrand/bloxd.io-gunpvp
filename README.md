# bloxd.io-gunpvp
## 要件定義
### 概要
2チームに分かれていて、チケット制。
銃でpvpする
### 試合の流れ
待機場所 ->
試合開始 ->
勝敗決定 ->
待機場所
### 通貨
円とコインがある
#### 円
勝敗決定時にもらえ、鯖に入り直しても保存される
#### コイン
試合中にもらえ、その試合限り
### キット
待機場所で円を使い購入でき、
購入済みのものを選択できる
試合開始時に選択したキットによりアイテムが付与される
### チケット
試合開始時に両チーム400になる
* 赤チームの死亡は赤チケット-1
* 青チームの死亡は青チケット-1
* 赤チームのビーコンの破壊は青チケット-10
* 青チームのビーコンの破壊は赤チケット-10
### 勝敗決定
敗北する条件は以下の通り
* 自チームのチケットが0になる
* 自チームの人数が0になる
* 試合開始から10分経過したとき、相手チームより自チームのチケットのほうが少ない

### 銃について
試合開始時にはM1911を付与する
試合中にコインを用いて銃を作業台でクラフトすることで入手できる
銃は同時に2つまでしか持てない
銃をドロップすると自動的に消え捨てられる
